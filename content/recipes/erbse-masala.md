# Zutaten

|               |                              |
|--------------:|:-----------------------------|
|     120 Gramm | Erbsen                       |
|         0,5-1 | Limette                      |
|      60 Gramm | Sonnenblumenöl oder Olivenöl |
|      40 Gramm | Sonnenblumenkerne            |
|               | Salz                         |
| 0,5 Teelöffel | Garam Masala                 |

---

# 1.

- [ ] Erbsen in heißem Wasser garen.
- [ ] Erbsen mit übrigen Zutaten pürieren.
- [ ] Mit Salz und Limettensaft abschmecken.
