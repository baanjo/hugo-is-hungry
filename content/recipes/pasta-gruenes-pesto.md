# Zutaten

|                |                |
|---------------:|:---------------|
|      100 Gramm | Cashews        |
|      300 Gramm | Nudeln         |
| (mindestens) 1 | Topf Basilikum |
|       70 Gramm | Olivenöl       |
|              2 | Knoblauchzehen |
|            0,5 | Limette        |
|                | Salz           |
|                | Pfeffer        |

---

# 1.

- [ ] Cashews in der Pfanne anrösten.
- [ ] Nudeln kochen.
- [ ] Cashews zusammen mit Basilikum, Olivenöl, Knoblauch, Limettensaft und nach Belieben Salz und Pfeffer zu einer
  feinen Masse mixen.
- [ ] Pesto zu den Nudeln geben.
